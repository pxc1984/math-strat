# math-strat

CLI-инструмент, который считает, **сколько надо минимум учить**, чтобы с высокой вероятностью набрать нужный балл.

Алгоритм перебирает стратегии, оценивает вероятность набора баллов и выбирает **самую дешевую**(субъективно). Цель: чтобы **p90 сдал**.

## Запуск

```bash
cargo run --release
```

При первом запуске программа прогревает кэш(у меня это занимает где-то секунд 5). Потом работает заметно быстрее.
